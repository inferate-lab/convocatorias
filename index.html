🕐 Configurar Escaneo Autónomo

1

Instalar Node.js

Descarga e instala Node.js desde nodejs.org si no lo tienes.

2

Instalar dependencias

cd inteligencia-fondos
npm install node-cron axios cheerio
3

Ejecutar el agente autónomo

node scheduler.js

El agente correrá escaneos cada 6 horas y guardará resultados en data/scans/

4

Abrir dashboard cada vez que entres

IRIS habrá trabajado mientras no estabas. Els resultados pre-clasificados te esperan.
